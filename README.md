# Stage-GRC-Assistant
# Assistant IA de Conformité & Gouvernance (GRC) sous Odoo

## À propos du projet
Ce projet a été développé dans le cadre de mon stage chez TRUSTIZI. Il vise à concevoir un assistant intelligent intégré à l'ERP Odoo pour automatiser la gestion de la conformité et la gouvernance des risques (GRC).

## Objectifs
- **Automatisation** : Simplifier le suivi des audits et des contrôles de conformité.
- **Support Décisionnel** : Utiliser l'IA pour analyser les risques de sécurité en temps réel.
- **Intégration** : Créer un module Odoo fluide et sécurisé.

## Stack Technique
- **ERP** : Odoo (Développement de module personnalisé)
- **Langage** : Python
- **Environnement** : Kali Linux (Machine virtuelle sur UTM)
- **Version Control** : Git & GitHub
